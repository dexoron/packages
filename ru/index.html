О реестре

Реестр бинарных пакетов проектов Dexoron. Готовые сборки для популярных дистрибутивов Linux — установка без компиляции из исходного кода.

Пакеты

Пакет Описание Платформы
dcr Утилита для управления C/C++ проектами в Cargo-like стиле

Установка

Выберите ваш дистрибутив для инструкции по установке.

Arch Linux

Добавьте репозиторий в /etc/pacman.conf и установите dcr:

1

Допишите в конец /etc/pacman.conf:

# /etc/pacman.conf
[dexoron]
SigLevel = Optional TrustAll
Server = https://dexoron.github.io/packages/archlinux/$arch
2

Обновите базу пакетов и установите dcr:

sudo pacman -Sy dcr

dcr --version
