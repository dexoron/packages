О репозитории

Репозиторий бинарных пакетов для проектов Dexoron. Готовые сборки для популярных дистрибутивов Linux — установка без компиляции из исходников.

Пакеты
Пакет Версия Описание Платформы
dcr v0.6.9 Утилита для управления C/C++ проектами в стиле Cargo
Arch Debian Fedora
Установка
1

Установите пакет с ключами для доверия к репозиторию:

sudo pacman -U https://pkg.dexoron.su/archlinux/x86_64/dexoron-keyring-20260528-1-any.pkg.tar.zst
2

Добавьте в конец /etc/pacman.conf:

# /etc/pacman.conf
[dexoron]
SigLevel = Required DatabaseOptional
Server = https://pkg.dexoron.su/archlinux/$arch
3

Обновите базу данных пакетов и установите:

sudo pacman -Sy dcr
1

Импортируйте ключ подписи:

curl -fsSL https://pkg.dexoron.su/dexoron.gpg | sudo tee /etc/apt/trusted.gpg.d/dexoron.asc > /dev/null
2

Добавьте репозиторий:

echo "deb [signed-by=/etc/apt/trusted.gpg.d/dexoron.asc] https://pkg.dexoron.su/debian ./" | sudo tee /etc/apt/sources.list.d/dexoron.list
3

Обновите список пакетов и установите:

sudo apt update && sudo apt install dcr
1

Добавьте репозиторий:

sudo curl -fsSL https://pkg.dexoron.su/fedora/x86_64/dexoron.repo -o /etc/yum.repos.d/dexoron.repo
2

Установите:

sudo dnf install dcr
